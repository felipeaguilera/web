# CV PDF downloads

Place generated PDFs here. Naming convention:

- cv-en-marketing.pdf   (EN · Marketing Strategist)
- cv-en-brand.pdf       (EN · Brand Consultant)
- cv-en-content.pdf     (EN · Content & Copy)
- cv-es-marketing.pdf   (ES · Estratega de Marketing)
- cv-es-brand.pdf       (ES · Consultor de Marca)
- cv-es-content.pdf     (ES · Estratega de Contenido)

Generate each one from cv-hub.html:
1. Open Job-Search-Hub/_generated/cv-hub.html in Chrome
2. Select language + role
3. File > Print > Save as PDF
4. Name it per the convention above and drop it in this folder
5. Push to GitHub to deploy
